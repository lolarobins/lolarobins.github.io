lolang compiler toolchain (current)

lolang is a compiler idea i have been planning to make for over two years (as of dec, 2024.) this is a project i mainly wanted to do just for fun, and as a learning experience.
my goal is to start with designing the base compiler, as well as adding a half-decent build system, and then slowly adding libraries and bindings while writing my other project ideas in lolang simultaneously.
currently, i am using C to write it, but as i intend to design this as a compiled language that can build to a static bin with only a c std lib dependency, i would like to write lolang, in lolang. :3
